# Персональный сайт-портфолио художника-постановщика

## Что это
Персональный сайт театрального художника-постановщика с портфолио работ, биографией и формой для связи. Заменяет PDF-файлы единой онлайн-точкой входа.

## Для кого
- Режиссёры и театры — ищут художника для постановки
- Продюсеры и менеджеры концертной индустрии — ищут художника для ивентов, выступлений артистов, концертов
- Ивент-агентства и бренды — ищут художника для мероприятий и коллабораций
- Частные заказчики — ищут художника для проекта

## Проблема
Портфолио существует только в виде PDF-файлов — неудобно делиться, сложно воспринимать, нет контекста о процессе и личности автора. Потенциальные работодатели и заказчики не могут быстро понять стиль работы, опыт и способ мышления художника, а также не имеют простого способа выйти на связь.

## Основные функции
- Галерея работ с полноэкранным просмотром фотографий
- Переключение языка сайта: русский / английский
- Скачать CV одной кнопкой (PDF)
- Контакт: email + мессенджер (Telegram или WhatsApp)
- Адаптивная вёрстка: десктоп и мобильная версия
- Тёмная тема оформления

## Страницы / экраны
1. **Главная** — первое впечатление: кто это, что делает, ключевые работы. *(Нужно отдельно продумать)*
2. **Сценография** — галерея спектаклей, концертов, ивентов и коллабораций с брендами; карточки проектов с фотографиями
3. **Как я работаю** — подход, этапы, эскизы, макеты, референсы; показывает мышление, а не только результат
4. **Резюме** — список театров, постановок, наград
5. **Обо мне** — биография, профессиональный путь, философия работы; можно упомянуть мастер-классы одной фразой
6. **Контакт** — email и мессенджер для запроса сотрудничества

## Чего НЕ будет (пока)
- Раздел «Преподавание» — добавим позже, когда будет готов отдельный материал с фото и примерами
- Раздел «Костюмы» — возможно, появится как отдельный раздел позже
- Пресса и рецензии
- Логотипы театров и площадок
- Арт-терапия и междисциплинарные проекты

## Технические заметки
- Двуязычность (RU / EN): переключатель языка, весь контент готовится в двух версиях
- Много фотографий: нужна оптимизация изображений для быстрой загрузки
- Полноэкранный просмотр: лайтбокс с листанием стрелками
- Тёмная тема: основная, светлую не планируем
- CV: PDF-файл обновляется вручную при изменениях
- Если фотографий по процессу мало — можно встроить их внутрь карточек проектов в Сценографии, без отдельной страницы
